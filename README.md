# SlideBack
一个仿 即刻APP 滑动返回效果的Demo

# 使用方法：
把上面代码中的SlideBackActivity.java，SlideBackView.java，view_slideback.xml三个文件拷贝到自己项目里
然后将自己项目中的Activity继承SlideBackActivity并重写slideBackSuccess方法即可
当触发滑动返回时，会调用slideBackSuccess方法
